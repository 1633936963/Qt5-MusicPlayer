# LightMusicPlayer
*——南京大学2019秋季学期 ”高级程序设计 “ 课程设计三*

基于Qt5开发的一个小巧精美的本地音乐播放器

![cover](README.assets/cover.png)



**已实现的目标：**

- 界面精美且支持自定义背景
- 支持播放的音乐格式：.mp3文件、.flac文件、.mpga文件
- 支持解析并展示歌词文件（.lrc文件）
- 支持解析歌曲信息（专辑、比特率、缩略图等等）
- 支持歌单管理
- 系统托盘可控制播放
- 数据库保存已添加的歌曲和歌单信息

  

**环境：**

Qt5 （Qt Creator 4.8.2 + Qt 5.9.8 MinGW 32bit）
